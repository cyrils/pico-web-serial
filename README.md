# Pico File Manager
A web based file manager for [Raspberry Pi Pico](https://www.raspberrypi.com/products/raspberry-pi-pico/). This library uses [Web Serial API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Serial_API) to communicate through USB serial port. It can read output log and supports basic file management and code editing. Works only with micropython firmware.

### How to use?

Connect Pico to your laptop using micro usb cable. Then visit the [demo page](https://cyrils.github.io/pico-file-manager/serial.html) in your chrome browser.
